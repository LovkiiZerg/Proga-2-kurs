# Proga-2-kurs
Здесь представлены лабораторные работы по программированию.

## Лабораторная №4
Лабораторная работа №4 - это оконное приложение на базе wxWidgets. Сам функционал реализуется через написанную ранее лабораторную работу №3.

Основные функции:

* добавление, редактирование и удаление пользователей
* сохранение данных в txt-файл
* сортировка по имени, количеству звонков и их стоимости
* подсчет средней цены и дохода
